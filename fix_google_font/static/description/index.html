Fix google font

Due to GDPR rulings, you can not let your website load google fonts. This module downloads google fonts to your server before passing them on to the user, protecting the user from sending any privacy protected data to a third party.

How it works

This module hooks into the generation of Odoo's fontend css and replaces the urls of google fonts with relative urls to your webserver. Additionally your webserver acts as a proxy parsing googles font page and pasing it to the enduser.

Info

This plugin only removes the calls to google for visitors. It does not remove the google fonts loading from the editor mode since this is only available for website administrators.

If you need this module for a different version of Odoo send a mail to info@bytist.de
